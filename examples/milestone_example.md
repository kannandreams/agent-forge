# Milestone Example

## Scenario

A team needs to ship a new API endpoint and coordinate the work across implementation, validation, and release.

## Applied Capability

- `skills/github/project-milestone/SKILL.md`
- `skills/github/issue-breakdown/SKILL.md`
- `templates/milestone_template.md`
- `workflows/milestone_execution.md`

## Result

The milestone is broken into a small set of workstreams, each with clear ownership, dependencies, and acceptance criteria.

## Notes

- Keep the milestone bounded.
- Track integration risk early.
- Verify the endpoint before release.
